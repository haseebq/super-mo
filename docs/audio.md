# Super Mo - Audio Direction

## Music
- Track 1: Title theme (bright, upbeat, 120-130 BPM)
- Track 2: World 1 loop (playful, bouncy, 100-110 BPM)

## SFX List
- Jump
- Stomp
- Coin pickup
- Rare collectible
- Damage / fail
- Level complete
- Menu select

## Style
- 16-bit inspired chiptune with cleaner mixing.
- Short, snappy effects; no long tails.

## Technical
- Use Web Audio for low latency.
- Preload and decode on start screen.
