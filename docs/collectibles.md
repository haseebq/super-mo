# Super Mo - Collectibles and Progression

## Coins
- Primary collectible, used for score.
- 100 coins grants 1 extra life (Phase 1 optional).

## Rare Collectible (Star Shard)
- One per level, optional challenge route.
- Collect 3 to unlock a bonus level.

## Checkpoints
- Mid-level checkpoint resets when player dies.

## Progression
- Linear progression by default.
- Optional level select once level 1 cleared.

## Rewards
- Visual feedback (sparkle + chime) on rare collectible.
- Simple end-of-level summary: coins + shards.
