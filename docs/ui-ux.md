# Super Mo - UI/UX Spec

## Screens
- Title screen: Start, Options (audio), Credits
- Pause screen: Resume, Restart Level, Quit
- Level complete: time, coins, shards

## HUD
- Top-left: Lives (icon + count)
- Top-center: Coins
- Top-right: Shards (0/1)

## Typography
- Pixel font for HUD (legible at small sizes)
- Title uses a chunky display font

## Controls Display
- Title screen shows: Arrows + Z (jump) + X (run)

## Accessibility
- Color contrast high for HUD
- Optional reduced shake (Phase 2)
