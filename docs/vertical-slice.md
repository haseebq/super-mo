# Super Mo - Vertical Slice Milestone

## Goal
A playable single level demonstrating core movement, one enemy, and basic UI/audio polish.

## Must-Haves
- Player movement (walk/run/jump) tuned to physics spec
- Level 1 layout implemented
- One enemy (Moomba) with stomp interaction
- Coins + rare collectible
- HUD with coins/lives/shards
- Title screen + pause
- Basic SFX + 1 music loop

## Acceptance Criteria
- Stable 60 FPS on desktop
- Level playable start to finish in 2-3 minutes
- No blocking bugs in collision or camera

## Deliverables
- HTML/JS/CSS scaffold
- Vector sprite placeholders or first-pass art
- Level data JSON for 1-1
